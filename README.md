# Vote
javaweb 大作业  
前后端分离的投票网站  
由 [vue3](https://v3.cn.vuejs.org/) + [Vite](https://vitejs.cn/) 开发  
由 [vote-javaweb](https://github.com/ltxhhz/vote-javaweb) 作为后端

* 随便写写，仅供学习参考
* 全部都是边想边写，没过脑子
* 许多应该考虑的问题
  * **数据加密传输**
  * **防刷票**
  * **危险操作二次确认**
* 所以不要直接用在生产环境
